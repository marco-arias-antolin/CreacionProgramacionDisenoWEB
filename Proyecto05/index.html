Primeros pasos:

En bootstrap se utilizan clases para organizar el contenido.

En vez de crear un estilo para la página web se usan las clases de bootstrap

Bootstrap es responsive y mobile first

Los puntos de ruptura se usan para el diseño responsive, pudiendo especificar un tamaño para cada tipo de dispositivo: breakpoints

En bootstrap hay distintos contenedores: containers

El sistema de rejilla de bootstrap tiene doce columnas: grid

El margen y el relleno

Las clases para el margen y el relleno se usan con el siguiente formato para tamaño xs:

  • {property}{sides}-{size}

Para el resto de tamaños (sm, md, lg, xl y xxl) se usa el formato:

  • {property}{sides}-{breakpoint}-{size}
Propiedad (property)
  • Margen (margin): m
  • Relleno (padding): p
Lados (sides)
  • t ➡️ (top) Arriba
  • b ➡️ (bottom) Abajo
  • s ➡️ (start) izquierda en LTR, derecha en RTL
  • e ➡️ (end) derecha en LTR, izquierda en RTL
  • x ➡️ (eje x) izquierda y derecha
  • y ➡️ (eje y) arriba y abajo
  • blank ➡️ No poner nada para aplicarlo a los cuatro lados
Tamaño (size)
  • 0 ➡️ Sin margen/relleno
  • 1 ➡️ $spacer * .25
  • 2 ➡️ $spacer * .5
  • 3 ➡️ $spacer
  • 4 ➡️ $spacer * 1.5
  • 5 ➡️ $spacer * 3
  • auto ➡️ automático
Ejemplos de margen y relleno
  • mt-0 ➡️ sin margen arriba
  • ms-1 ➡️ margen izquierdo en LTR
  • py-2 ➡️ relleno en el eje y
  • p-3 ➡️ relleno en los cuatro lados
  • mx-auto ➡️ margen en el eje x automático

Ejemplo responsive de columnas

Las columnas ocupan

  • sm (≥576px) el 100%
  • md (≥768px) el 50%
  • lg (≥992px) y más grandes ocupan 2-8-1-1 (total 12)
