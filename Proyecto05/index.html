Primeros pasos:

En bootstrap se utilizan clases para organizar el contenido.

En vez de crear un estilo para la página web se usan las clases de bootstrap

Bootstrap es responsive y mobile first

Los puntos de ruptura se usan para el diseño responsive, pudiendo especificar un tamaño para cada tipo de dispositivo: breakpoints

En bootstrap hay distintos contenedores: containers

El sistema de rejilla de bootstrap tiene doce columnas: grid

El margen y el relleno también tienen clases:

  • Margen (margin): m-0, ..., m-5
  • Relleno (padding): p-0, ..., p-5
